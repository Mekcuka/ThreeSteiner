Настройки

options.euclid_steiner_candidates — точки ESMT через efst|bb (env GEOSTEINER_HOME); без бинарников — warning.

Зоны ban / penalty

terrain.zones[].buffer_m — отступ трассы (оранжевый контур на карте). allow_terminal_in_zone_buffer — терминал в кольце buffer разрешён; в core ban (красный) — по-прежнему 422.

Post-process

«Чистое Steiner»: hub/attach выкл., spacing и repel = 0. Круги: голубой — connector_max_km (если enforce), оранжевый — steiner_radius_km (repel).

Редактор

Оба поля — для нового терминала; 0° север, 90° восток.
